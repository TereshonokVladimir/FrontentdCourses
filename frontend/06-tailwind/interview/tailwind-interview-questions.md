# Interview Questions: Tailwind CSS

## Beginner

1. What is Tailwind CSS? How does it differ from Bootstrap?
2. What does "utility-first" mean?
3. How do you add padding to an element? Margin?
4. What is the difference between `p-4` and `px-4`?
5. How do you make a flexbox container? How do you center items?
6. What is the purpose of the `gap` utility?
7. How do you apply styles on hover? On focus?
8. What is the difference between `hidden` and `invisible`?
9. How do you make a responsive layout (e.g., 1 column on mobile, 3 on desktop)?
10. What is the `rounded` utility used for?
11. How do you set a background color? Text color?
12. What is the Tailwind config file used for?
13. How do you add a custom color to your theme?
14. What is the `@apply` directive? When would you use it?
15. How do you enable dark mode in Tailwind?

## Intermediate

1. Explain the Tailwind spacing scale. What does `p-4` translate to?
2. What is the difference between `space-x-4` and `gap-4`?
3. How do you use the `group` and `group-hover` utilities?
4. What is the `peer` utility? Give an example.
5. How do you create a responsive design with Tailwind? What is mobile-first?
6. What is the difference between `max-w-md` and `w-full max-w-md`?
7. How do you add a focus ring for accessibility?
8. What is the difference between `focus:` and `focus-visible:`?
9. How do you use arbitrary values in Tailwind? Give an example.
10. What is the purpose of `@layer`? What layers exist?
11. How do you configure dark mode? What is the difference between `media` and `class`?
12. What is the `container` utility? How do you use it?
13. How do you add a custom utility in the config?
14. What is the difference between `theme.extend` and `theme`?
15. How do you use the `ring` utilities? When would you use them instead of `border`?
16. What is the `divide` utility used for?
17. How do you create a custom animation in Tailwind?
18. What is the `prose` class? Which plugin provides it?
19. How do you prevent layout shift when loading images?
20. What is the purpose of the `content` array in the Tailwind config?

## Advanced

1. Explain how Tailwind's JIT (Just-In-Time) compiler works. What are the benefits?
2. How does Tailwind handle purging unused styles in production?
3. What is the difference between `min-w-0` and `min-w-min`? When is `min-w-0` critical?
4. How would you implement a design system with Tailwind? What would you put in the config?
5. Explain the cascade and specificity when using @apply with @layer.
6. How do you use CSS variables with Tailwind for runtime theming?
7. What is the `safelist` option in the config? When would you use it?
8. How do you handle dynamic class names (e.g., from a variable)? What are the pitfalls?
9. What is the difference between `backdrop-blur` and `blur`?
10. How would you optimize Tailwind for a large project? What strategies?
11. What is the `important` option in the config? When would you use it?
12. How do you integrate Tailwind with CSS-in-JS (e.g., styled-components)?
13. What is the `prefix` option? When would you use it?
14. How do you handle RTL (right-to-left) layouts with Tailwind?
15. What is the difference between `transition` and `transition-all`? Which is better for performance?
16. How do you support `prefers-reduced-motion` with Tailwind?
17. What is the `@tailwindcss/forms` plugin? What does it do?
18. How would you structure a component library using Tailwind?
19. What are the trade-offs of using @apply vs inline utilities?
20. How do you debug Tailwind when a class doesn't seem to work?
