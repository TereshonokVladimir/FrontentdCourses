# Interview Questions: React

## Beginner

1. What is React? What problem does it solve?
2. What is the difference between a component and an element?
3. What is JSX? How does it differ from HTML?
4. What are props? Can a component change its props?
5. What is state? How is it different from props?
6. What is the difference between `useState` and `useRef`?
7. When would you use a key prop? Why is it important?
8. What is the virtual DOM? How does React use it?
9. What is the difference between controlled and uncontrolled components?
10. How do you handle events in React?
11. What is conditional rendering? Give an example.
12. What are React hooks? Name three built-in hooks.
13. Why can't you call hooks inside loops or conditions?
14. What does `useEffect` do? When does it run?
15. What is the difference between `createElement` and JSX?

## Intermediate

1. Explain the React component lifecycle (mount, update, unmount) in functional components.
2. What is the purpose of `useCallback`? When would you use it?
3. What is `useMemo`? How does it differ from `useCallback`?
4. What is `React.memo`? When does it prevent re-renders?
5. Explain lifting state up. When would you use it?
6. What is Context? When would you use it instead of props?
7. What are custom hooks? What rules do they follow?
8. How do you prevent unnecessary re-renders in React?
9. What is the difference between `useEffect` with empty deps and with no deps?
10. How do you handle forms in React? Controlled vs uncontrolled.
11. What is a ref? When would you use `useRef` vs `useState`?
12. Explain the rules of hooks. Why do they exist?
13. What is prop drilling? How can you avoid it?
14. What is composition vs inheritance in React?
15. How do you fetch data in React? What about cleanup?
16. What is `forwardRef`? When would you use it?
17. What is `useImperativeHandle`? Give a use case.
18. What are Fragments? When would you use them?
19. What are Portals? Give an example.
20. How do you handle errors in React? What are error boundaries?

## Advanced

1. Explain how React's reconciliation algorithm works. What is the diffing strategy?
2. What is the purpose of `key` in reconciliation? Why not use index when the list can reorder?
3. Explain the React fiber architecture. What is a fiber?
4. What is Concurrent Mode (or Concurrent Features)? How does it change rendering?
5. What is Suspense? How does it work with data fetching?
6. How does `useEffect` cleanup work? Why is it important for subscriptions?
7. What causes stale closures in React? How do you fix them?
8. Explain batching in React. When are state updates batched?
9. What is the difference between `useLayoutEffect` and `useEffect`?
10. How would you implement a virtualized list? What are the key considerations?
11. What is Server Components (React Server Components)? How do they differ from client components?
12. How does React handle event delegation? What are synthetic events?
13. What is the difference between `createPortal` and normal render?
14. How would you implement code splitting in React?
15. What is hydration? What happens during hydration?
16. Explain React's scheduling and prioritization. What is a lane?
17. How do you test React components? What is the testing pyramid?
18. What is the difference between `getDerivedStateFromError` and `componentDidCatch`?
19. How would you optimize a React app with many re-renders?
20. What is Strict Mode? What does it do and why?
