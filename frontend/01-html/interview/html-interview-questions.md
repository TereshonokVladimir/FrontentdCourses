# Interview Questions: HTML

## Beginner

1. What is the purpose of `<!DOCTYPE html>`?
2. What is the difference between `<div>` and `<span>`?
3. When should you use `<strong>` vs `<b>`?
4. What is the `alt` attribute used for on images?
5. What is the difference between `<ul>` and `<ol>`?
6. Why is the `lang` attribute important on the `<html>` element?
7. What does the `required` attribute do on form inputs?
8. What is the purpose of the `<meta charset="UTF-8">` tag?
9. How do you create a link that opens in a new tab? What security consideration should you add?
10. What is the difference between `<th>` and `<td>`?
11. What semantic element would you use for the main content of a page?
12. Why should you use `<label>` with form inputs?
13. What is the purpose of the `name` attribute on form inputs?
14. How do you create a same-page anchor link?
15. What is the difference between `<article>` and `<section>`?

## Intermediate

1. Explain the difference between `<header>` and `<h1>`. Can a page have multiple headers?
2. What is the purpose of the `scope` attribute on `<th>`? When is it required?
3. How does the `pattern` attribute work for form validation? What should you pair it with?
4. What is the `loading="lazy"` attribute on images? When should you use it?
5. Explain the purpose of `rel="noopener noreferrer"` when using `target="_blank"`.
6. What is the difference between `method="GET"` and `method="POST"` in forms?
7. How do you provide multiple image formats for different browsers? What element do you use?
8. What is the purpose of the `sandbox` attribute on iframes?
9. When would you use `<figure>` and `<figcaption>` instead of a plain `<img>`?
10. What is the `for` attribute on `<label>`? Why is it important?
11. Explain the purpose of `data-*` attributes. How do you access them in JavaScript?
12. What is the difference between `aria-label` and `aria-labelledby`?
13. When should you use `alt=""` (empty string) on an image?
14. What is the purpose of the `<picture>` element vs `<img>` with `srcset`?
15. How do you add captions/subtitles to a video element?
16. What is the purpose of the `datetime` attribute on the `<time>` element?
17. Why should you set `width` and `height` on images even when using CSS?
18. What is the difference between `<input type="submit">` and `<button type="submit">`?
19. How does the browser handle form validation? What happens when validation fails?
20. What is the purpose of the `preload` attribute on video and audio?

## Advanced

1. Explain the browser rendering pipeline and how HTML parsing fits into it.
2. What is the DOM? How does it differ from the HTML source?
3. What is "quirks mode" vs "standards mode"? How does DOCTYPE affect this?
4. Explain the same-origin policy and how it affects iframes.
5. What are the accessibility implications of using `role="button"` on a div? What must you implement?
6. How does the browser build the accessibility tree from HTML?
7. What is the difference between `defer` and `async` on script tags? When would you use each?
8. Explain the purpose of `rel="preconnect"` and `rel="preload"`. How do they affect performance?
9. What is the Constraint Validation API? How can you customize validation messages?
10. How does the `form` attribute allow inputs to be associated with a form outside their parent?
11. What is the purpose of the `enterkeyhint` attribute on inputs?
12. Explain how `contenteditable` works and its accessibility considerations.
13. What is the `inert` attribute? When was it added and what does it do?
14. How do you implement a focus trap in a modal dialog for accessibility?
15. What is the difference between `aria-hidden` and `hidden`? When would you use each?
16. Explain the `popover` API and the `popover` attribute. How does it differ from a custom modal?
17. What is the `dialog` element? How does it handle focus and escape key?
18. How does the browser handle invalid HTML? What is "tag soup" parsing?
19. What is the purpose of `fetchpriority="high"` on images?
20. Explain the relationship between HTML semantics and SEO. How do search engines use heading hierarchy?
