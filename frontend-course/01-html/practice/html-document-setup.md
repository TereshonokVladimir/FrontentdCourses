# Practice: HTML Document Setup

## Tasks

### Task 1
Create a minimal valid HTML5 document with DOCTYPE, html, head, and body. Include charset and viewport meta tags. Add a title: "My First Page".

### Task 2
Extend the document with a meta description and Open Graph title. Add a favicon link (use a placeholder path). Ensure the html element has `lang="en"`.

### Task 3
Create a document that includes a skip link at the top (linking to `#main`), a header with a nav, a main section with id="main", and a footer. Use semantic elements only.

### Task 4
Build a document template suitable for a multi-page site. Include: charset, viewport, title, description, canonical link, and Open Graph meta. Add placeholder sections for header, main, and footer with comments marking where content goes.

### Task 5
Create a production-ready HTML boilerplate with: DOCTYPE, lang, charset, viewport, theme-color, title, description, robots meta, canonical, og:title, og:description, og:image, favicon, and preconnect for a CDN. Add skip link and semantic structure.
