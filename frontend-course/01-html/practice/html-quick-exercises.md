# Practice: Quick HTML Exercises

## Tasks

### Task 1
Create a simple "About Me" card: your name as h2, a paragraph, and three bullet points (hobbies or skills). Use ul for the list. Add a small profile image with alt.

### Task 2
Build a navigation menu with five items. Make the current page item non-clickable (span) and add aria-current="page". Style the structure with semantic HTML only.

### Task 3
Create a "Sign up" form with: email, password, confirm password. Use type="email" and type="password". Add a "Show password" checkbox (structure only). Include a link to "Terms of Service" and a submit button.

### Task 4
Build a simple table: two columns (Name, Score), three rows of data. Add a caption. Use thead and tbody. Make the table responsive-friendly (add a wrapper and consider scope).

### Task 5
Create a "Newsletter signup" block: heading, short description, email input, subscribe button, and a note about privacy. Use a form with proper labels. Add a link to the privacy policy. Wrap in a section or aside with an appropriate heading.
