# Interview Questions: JavaScript

## Beginner

1. What is the difference between `let`, `const`, and `var`?
2. What are the primitive types in JavaScript?
3. What is the difference between `==` and `===`?
4. What is a function? How do you define one?
5. What is the difference between `null` and `undefined`?
6. What does `typeof` return for an array?
7. How do you add an item to an array? How do you remove one?
8. What is the difference between `map` and `forEach`?
9. What is a callback function?
10. What does `this` refer to in a regular function vs an arrow function?
11. What is the purpose of `return` in a function?
12. How do you access a property of an object?
13. What is the difference between `let` and `const`?
14. What is a Promise?
15. How do you handle errors in JavaScript?

## Intermediate

1. Explain the event loop. What is the difference between macrotasks and microtasks?
2. What is a closure? Give an example of a practical use case.
3. What is the difference between `async/await` and Promises?
4. How does `this` work? When would you use `bind`, `call`, or `apply`?
5. What is the difference between shallow copy and deep copy? How do you create each?
6. Explain `map`, `filter`, and `reduce`. When would you use each?
7. What is hoisting? How does it differ for `var` and `let`?
8. What is the difference between `addEventListener` and `onclick`?
9. What is event delegation? Why is it useful?
10. How do you prevent the default behavior of an event?
11. What is the purpose of `use strict`?
12. What is destructuring? Give examples for arrays and objects.
13. What is the spread operator? What is the rest parameter?
14. How does `fetch` work? What does it return?
15. What is the difference between `localStorage` and `sessionStorage`?
16. What is a pure function?
17. What is the difference between `slice` and `splice`?
18. How do you handle asynchronous errors?
19. What is the prototype chain?
20. What is the difference between `call` and `apply`?

## Advanced

1. Explain the JavaScript execution context (creation phase, execution phase).
2. What is the difference between the DOM and the virtual DOM?
3. How does garbage collection work in JavaScript?
4. What is a memory leak? How can you cause one in JavaScript?
5. Explain the module system (CommonJS vs ES modules).
6. What is the purpose of `Symbol`? When would you use it?
7. What is the difference between `Object.create` and `new`?
8. How does `Promise.all` work? What happens if one Promise rejects?
9. What is the Temporal Dead Zone?
10. Explain the difference between `for...in` and `for...of`.
11. What is a WeakMap? When would you use it?
12. How does `requestAnimationFrame` work?
13. What is the difference between `setTimeout` and `setInterval`?
14. How do you implement a debounce function?
15. What is the difference between `preventDefault` and `stopPropagation`?
16. What is the event bubbling and capturing phase?
17. How does `JSON.stringify` handle circular references?
18. What is the difference between `async` and `defer` on script tags?
19. Explain the same-origin policy and CORS.
20. What is the difference between a shallow comparison and a deep comparison? How would you implement a deep equality check?
