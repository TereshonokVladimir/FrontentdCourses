# Interview Questions: CSS

## Beginner

1. What is the difference between `margin` and `padding`?
2. What does `box-sizing: border-box` do?
3. How do you center an element horizontally?
4. What is the difference between `display: none` and `visibility: hidden`?
5. What are the values of the `position` property?
6. How do you make a link change color on hover?
7. What is the difference between `class` and `id` in terms of CSS?
8. How do you include an external stylesheet?
9. What does `float` do? When would you use it today?
10. What is a pseudo-class? Give two examples.
11. How do you select all paragraphs inside a div?
12. What is the `z-index`? When does it work?
13. What does `!important` do?
14. How do you make an element take full viewport height?
15. What is the difference between `em` and `rem`?

## Intermediate

1. Explain the CSS cascade and specificity. How is the winner determined?
2. What is the difference between Flexbox and Grid? When would you use each?
3. How does `flex: 1` work? What are `flex-grow`, `flex-shrink`, and `flex-basis`?
4. Explain `grid-template-columns: repeat(auto-fill, minmax(250px, 1fr))`.
5. What triggers a layout (reflow) vs a repaint? Which properties are safe to animate?
6. How do you implement a mobile-first responsive design?
7. What is the difference between `:focus` and `:focus-visible`?
8. How do CSS custom properties (variables) work? How do they differ from SASS variables?
9. What is a stacking context? What creates one?
10. Explain `position: sticky`. What can prevent it from working?
11. What does `overflow: hidden` do to `position: sticky` descendants?
12. How do you create a responsive image that scales with the container?
13. What is the purpose of `prefers-reduced-motion`?
14. How does `object-fit` work on images?
15. What is the difference between `transform` and `position` for moving elements?
16. How do you create a CSS-only dropdown or accordion?
17. What is BEM? Give an example of the naming convention.
18. How does `gap` work in Flexbox and Grid?
19. What is the difference between `@media` and `@container` queries?
20. How do you prevent layout shift (CLS) when loading images?

## Advanced

1. Explain the browser rendering pipeline. Where does CSS fit in?
2. What is the difference between the CSSOM and the DOM?
3. How does the browser match selectors? Why does it go right-to-left?
4. Explain containment: `contain: layout`, `contain: paint`, `contain: size`.
5. What is `content-visibility`? How does it improve performance?
6. How do `@layer` and cascade layers work?
7. Explain `@supports` and when you'd use it.
8. What is the difference between `inherited` and `non-inherited` properties?
9. How does `currentColor` work?
10. What is the `:has()` selector? What are its performance implications?
11. Explain `scroll-snap-type` and `scroll-snap-align`.
12. How do you implement a focus trap for a modal (conceptually)?
13. What is `aspect-ratio`? How does it help with layout?
14. Explain `@container` queries vs media queries.
15. What is the difference between `width: 100%` and `width: 100vw`?
16. How does `will-change` work? When should you use it?
17. Explain the `all` property and its values (`initial`, `inherit`, `unset`, `revert`).
18. What is the `inert` attribute and how does it affect styling?
19. How do you debug "why isn't my style applying?"
20. Explain the relationship between CSS and accessibility. What styling choices affect a11y?
